Length Optimizer


The Application consists of two main windows input window and reuslt display window
In input window user need to add following items --
1) Part Name/Number , Required length, Quantity of lengtth
2) Then click on arrange button where it is grouped according to the part name/number
3) Add maximum element length availbale in the market/available
4) Add Cutting Allowance reqiuired (min\default = 5mm)
5) Result can viewed by clicking 'Show' button of the respective group
